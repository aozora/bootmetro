Enabling selection mode

To enabling the toggle selection simply use a script like this:

   $('#listview-grid-demo').on('click', '.mediumListIconTextItem', function(e){
      e.preventDefault();
      $(this).toggleClass('selected');
   });
In the next version there will be a jQuery plugin to expand these functionality.
