BootMetro

Metro style web framework

Simple and flexible HTML, CSS, and Javascript for web apps that wants to use the "Windows 8 App Store" style, without the need to run on Windows 8.

Features

BootMetro is built on top of Twitter Bootstrap and inspired by the Metro UI CSS by Sergey Pimenov

Create your Hub

Give more focus to your content. With a Metro style app you give users immediately focus on the primary infos, increasing the usability and reducing the distractions.

Use the same tile templates as defined for the original Windows 8 MetroUI.

Actions

Use the ApplicationBar to show contextual icon action.

Navigation

Implement the navigation in pure MetroUI style, use back-button and the header menu to provide simple ways to navigate to other pages.

Charms

Create charms panels to show contextual settings.

Font Icons

With the awesome IcoMoon font icon set, you can use hundreds of font icons with any color and any dimension.
